Linux
Arch Arch is my daily OS. So why am I not using Windows, like everybody else? Well first of all I found Windows has just transformed into a huge and complex solution, that needs a lot of RAM . So, a friend(
Luis
)told me, how little ram his Linux laptop was using in idle. I noticed this was 1/8th of what my windows PC was using, so I understood that something had to be done. I installed Debian with xfce and was surprised by the control I had gained. My PC didn’t take years to boot up and everything was so clean. Of course, the first few weeks were different, but knowing that I am get used to it. I can confidently say that I only use Windows to do GPU intensive things. Plus, Linux just improves my workflow and boosts my productivity, since everything is so quickly accessible through the terminal. A few months after my friend introduced me to Linux, I noticed, that the GNOME desktop environment is very clean. So I tried to install it, but found out, that a lot of applications came with it and that was just too much. It also compromised the reason I switched to Linux. I wanted to get rid of the bloat. But after some more discussion and research I learned that found out that while installing arch, you could choose to not install all of these applications. This was great, plus it was a major breath of fresh air, escaping Debian’s repositories and using arch user repos. Since then, I have been on Arch Linux.
Debian Debian - as I already mentioned Debian was my first Linux OS. I am very thankful that I had spent some time there, since most of the world’s servers run on Debian. Exactly like the servers my friend and me  set up in our school also running on Debian. Therefore, this was easy on the PC side. It also provided an important step-stone from Windows to Arch. Without it I would have probably stepped back into Windows “hell.”
